# Getting started with ES - By Compose

- [Code and TOC](https://github.com/compose-ex/petitioneering)

## Getting started

```Bash
$ uname
Darwin
$ node --version
v6.9.2
$ npm --version
3.10.9
$ npm install -g yarn
$ yarn --version
0.21.3

$ yarn install
```

```Bash
$ ES_HOSTS="localhost:9200" node index.js
```
